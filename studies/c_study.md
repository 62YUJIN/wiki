## 1주차
- 2017.03.27
- 출석: 김규희, 김문수, 박찬우, 장은애, 허수진
- 진도: ~ ASCII
- 과제: 삼각형의 밑변과 높이를 double형 정수로 입력받아 삼각형의 넓이를 소수점 셋째자리까지 출력

과제 코드
```c
#include <stdio.h>

int main(void)
{
	double x,y;
	double area;

	printf("밑변을 입력하세요: ");
	scanf("%lf", &x);

	printf("높이를 입력하세요: ");
	scanf("%lf", &y);

	area = (x*y)/2;

	printf("넓이는 %.3f입니다.\n", area);

	return 0;
}
```

```c
#include <stdio.h>

int main(void)
{
	int sum = 0;
    int i = 0;

	for(i=0; i<100; i++)
    {
    	sum += 1;
    }
    
 	return 0;
}
```

예제2
```c

int main(void)
```
